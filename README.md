# lichtkunst.cc

Website von Istvan Seidel – Lichtkünstler. Sky Art, Geometrie aus Licht,
NFTs auf der TON Blockchain.

## Tech-Stack

- [Astro](https://astro.build/) — Static Site Generator
- [Tailwind CSS v4](https://tailwindcss.com/) — Styling
- Hosting: [Cloudflare Pages](https://pages.cloudflare.com/)

## Lokal entwickeln

```bash
npm install
npm run dev
```

Site läuft auf `http://localhost:4321`.

## Build

```bash
npm run build
```

Output in `dist/`.
